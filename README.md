# Network-Application-Final-Project

操作步驟：
1. 安裝所有需要的函式庫：pytrends、pandas、flask
2. 在資料夾中打開終端執行：python app.py
3. 打開本地的瀏覽器，網址為：http://127.0.0.1:5000/，即可使用功能
